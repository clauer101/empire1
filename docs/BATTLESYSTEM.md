Die Battles sollen folgende neue funktion erhalten:
Verschiedene user sollen gemeinsam einen defender angreifen können.

Aktueller Stand:
* user greifen getrennt von einander an
* battle service startet eine battle pro armee
* towers schießen in jeder battle unabhängig voneinander
* spectators subscriben zu jeweisl einer battle

Gewünschter stand:
* battles finden pro defender gleichzeitig statt wenn sie zeitgleich starten
* wenn eine battle startet während bereits eine andere battel läuft joined der zwei (oder meherer) angreifer einer battle.
* Alle einheiten sidn teil derselben battel
* towers müssen gegen alle critters gelichzeitig kämpfen, schießen aber in ihrer ursprünglichen rate
* spectatos subscriben nicht mehr zu einer sepzifischen battle sondern zu einem verteildigenden empire
* loot wird zwischen allen anfreifern gleich geteilt.
* artefacte wird zufälig ausgewürfelt, artefakt bekommt nur einer bei erfolg

* erzeuge neue unit tests für den gewünschten zielzustand
