🔧 Dev Tools

Developer tools for the Empire project

Economy
⚡ Effects Overview All effect keys by building and category. Filter effects, inspect values and debug. via API ⏱ Era Effects Era-level effect modifiers: inspect and tune per-era bonuses and multipliers. via API ✨ Artifacts Edit artifact definitions: name, type, passive effects. Save directly to artifacts.yaml. via API 👑 Rulers Configure rulers: name, type, skills Q/W/E (5 levels) and R (3 levels) with effect dicts. via API 🎛️ Sigmoid Tuner Interactively calibrate price curves for citizens, tiles, waves and critters. Copy-ready values. Local ⏱ Effort Tuner Adjust build and research times per era. Time estimates with speed boosts. Save directly. via API 🔧 Workshop Config Tower and unit upgrade bonuses (% per level) and base costs per era. Live reload. via API 🐾 Critters Inspect all critter definitions: stats, slots, era, animations. via API 📊 Unit Scatter All critters and towers as a scatter plot. X-axis: era, Y-axis: power index. Color-coded by class. via API ⚖️ Balance Overview Critters vs. towers per era: DPS, unlock effort, counter pairs and balance warnings. via API
Battle
🌊 AI Wave Composer Define attack waves per era, set triggers, configure travel and siege times. via API 🤖 AI Army Generator Auto-generate AI armies based on era, tower value, path length and buildings. Tune weights via sliders. via API ⚔ Send AI Army Manually send an AI army to an empire. Empire analysis with power index, tower costs and path length. via API 📈 AI Battle Log Scatter plot of all AI attacks: tower gold vs. era. Green = defender wins, red = AI wins. via API ⚔ Sim Map Simulate battles on a map: place towers, spawn waves, observe combat resolution. via API 🗺 Map Overview Hex maps of all empires: power index, path length, tower stats. Rename and delete. via API 🖼 Sprite Preview Animate sprite sheets, switch directions, tune FPS and size. All frames as overview. Local
Helpers
📊 Server Status Live overview: connections, all empires, life, culture, citizens and active attacks. via API 🗄️ Database Admin Manage user accounts: create, delete, reset password. via API 🔁 Restart Server Save state and restart the gameserver process. All clients briefly lose connection. via API 🌍 Global Map World-level map view of all empires and their positions. via API
